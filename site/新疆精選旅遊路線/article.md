根據導遊小雅攻略整理，以烏魯木齊與喀什為核心，梳理出四條最經典的新疆旅行環線。

## 行前自駕提醒（全線適用）

- 導航建議用**高德地圖（Amap）**，GCJ-02 座標最精準
- 高德地圖座標格式為**經度,緯度**（與 Google Maps 相反）
- **獨庫公路**每年 6 月中旬至 10 月初開放，出發前查即時路況
- 冬季可玩阿勒泰**將軍山滑雪場**（12–2月粉雪期最佳）

## 路線一：北疆大環線（10–12天 首選全景遊）

**推薦對象**：第一次去新疆、假期充裕、想一次收齊沙漠 / 湖泊 / 神仙村落 / 最美公路

**最佳季節**：6 月 ~ 10 月

**路線**：烏魯木齊 → S21沙漠公路 → 阿勒泰/禾木/喀納斯 → 五彩灘 → 魔鬼城 → 奎屯 → 賽里木湖 → 伊寧 → 夏塔/喀拉峻/那拉提 → 獨庫公路/唐布拉 → 烏魯木齊

### 逐日行程
- **Day 1** — 烏魯木齊出發，沿 S21 沙漠公路橫跨古爾班通古特沙漠，抵達阿勒泰市（冬季則為將軍山滑雪場）
- **Day 2** — 阿勒泰 → 禾木風景區，圖瓦人原始村落，木屋散步
- **Day 3** — 禾木清晨看晨霧 → 喀納斯景區遊覽神仙灣、月亮灣、臥龍灣
- **Day 4** — 喀納斯 → 布爾津五彩灘，欣賞額爾齊斯河畔夕陽雅丹地貌
- **Day 5** — 布爾津 → 烏爾禾世界魔鬼城（雅丹群）→ 奎屯市
- **Day 6** — 奎屯 → 賽里木湖景區（大西洋最後一滴眼淚）
- **Day 7** — 賽里木湖 → 果子溝大橋 → 伊寧市（藍色小鎮、人文風情）
- **Day 8** — 伊寧 → 夏塔古道（木札爾特冰川）→ 特克斯八卦城
- **Day 9** — 特克斯 → 喀拉峻（立體草原）或 那拉提（空中草原）
- **Day 10** — 那拉提 → 唐布拉百里畫廊 → 獨庫公路縱跨天山
- **Day 11–12** — 獨庫公路北段 → 返回烏魯木齊

### 亮點景點
- 禾木 / 喀納斯：人間仙境，晨霧中的圖瓦族村落
- 五彩灘：傍晚日落最佳，五彩岩層倒映額爾齊斯河
- 魔鬼城（烏爾禾）：壯麗雅丹地貌，風蝕奇景
- 賽里木湖：碧藍如鏡，高山天湖
- 那拉提草原：騎馬看天山
- 獨庫公路：僅 6月中–10月開放，穿越天山腹地

## 路線二：伊犁環線（6–8天 草原花海）

**推薦對象**：熱愛拍照、綠色草原、薰衣草花海的網美與攝影愛好者

**最佳季節**：5 月 ~ 8 月（6月中旬薰衣草花季最佳）

**路線**：烏魯木齊 → 賽里木湖 → 果子溝大橋 → 霍城薰衣草莊園 → 伊寧 → 夏塔 → 喀拉峻 → 那拉提 → 唐布拉 → 獨庫公路 → 烏魯木齊

### 逐日行程
- **Day 1** — 烏魯木齊出發，直奔賽里木湖景區，環湖欣賞高山藍湖
- **Day 2** — 賽里木湖 → 果子溝大橋 → 霍城解憂公主薰衣草園 → 伊寧市
- **Day 3** — 伊寧 → 昭蘇夏塔古道景區，雪山與森林交織
- **Day 4** — 昭蘇 → 喀拉峻景區（立體草原，特克斯八卦城附近）
- **Day 5** — 特克斯 → 那拉提旅遊風景區，騎馬漫步空中草原
- **Day 6** — 那拉提 → 尼勒克唐布拉風景區（百里畫廊慢活）
- **Day 7–8** — 唐布拉 → 獨庫公路（北段）→ 穿越天山 → 返回烏魯木齊

### 亮點景點
- 賽里木湖：開場震撼，碧湖映雪山
- 果子溝大橋：雙層高架橋穿越峽谷
- 霍城薰衣草莊園：中國最大薰衣草產區
- 喀拉峻草原：層疊草原如調色盤
- 那拉提：氈房體驗哈薩克族文化
- 唐布拉百里畫廊：峽谷溪流並行

## 路線三：南疆小環線（6–7天 高原人文）

**推薦對象**：深度旅人，熱愛異域人文與帕米爾高原極限公路

**最佳季節**：
- 4月 — 帕米爾滿山杏花
- 7–9月 — 西域最甜美的瓜果
- 10–11月 — 金黃帕米爾胡楊林

**路線**：喀什 → 白沙湖 → 卡拉庫里湖 → 慕士塔格峰 → 塔縣 → 盤龍古道 → 班迪爾藍湖 → 返回喀什

### 逐日行程
- **Day 1** — 抵達喀什古城，漫步百年老茶館，體驗維吾爾異域風情
- **Day 2** — 喀什出發，沿中巴公路上攀帕米爾高原，抵達白沙湖景區（藍湖 + 銀沙山）
- **Day 3** — 白沙湖 → 卡拉庫里湖景區，近距離仰望冰山之父慕士塔格峰
- **Day 4** — 卡拉庫里湖 → 抵達邊境高山縣城塔什庫爾干（塔縣，需邊防證）
- **Day 5** — 塔縣 → 挑戰盤龍古道（數百個彎道，走過所有彎路，從此儘是坦途）
- **Day 6** — 盤龍古道 → 班迪爾藍湖（Tiffany藍，下坂地水庫天然奇景）
- **Day 7** — 從高原下山，返回喀什古城圓滿結束

### 亮點景點
- 喀什古城：千年絲路重鎮，維吾爾建築迷宮
- 白沙湖：帕米爾純白沙山，極致純淨
- 卡拉庫里湖：慕士塔格峰倒影，世界頂級風景
- 慕士塔格峰：「冰山之父」7,546m，入門級冰川徒步
- 盤龍古道：連續急彎盤旋，視覺震撼
- 班迪爾藍湖：極致 Tiffany 藍

## 路線四：東疆短線（3–5天 絲路荒漠）

**推薦對象**：假期較短，想體驗火焰山傳奇與浩瀚沙漠

**最佳季節**：5–6月 / 9–10月

**路線**：烏魯木齊 → 吐魯番（火焰山）→ 鄯善（庫木塔格沙漠）→ 哈密（魔鬼城）→ 巴里坤草原 → 返回烏魯木齊

### 逐日行程
- **Day 1** — 烏魯木齊出發，抵達吐魯番市，探訪坎兒井、葡萄溝
- **Day 2** — 吐魯番 → 火焰山景區（地表溫度 70°C，西遊記經典場景）→ 鄯善縣
- **Day 3** — 鄯善 → 庫木塔格沙漠（世界上離城市最近的浩瀚沙海）
- **Day 4** — 鄯善向東 → 哈密市 → 哈密五堡魔鬼城（獨特雅丹群）
- **Day 5** — 哈密 → 巴里坤高山草原 → 沿連霍高速返回烏魯木齊

### 亮點景點
- 吐魯番火焰山：夏天地表 70°C，西遊記聖地
- 庫木塔格沙漠：鄯善城市邊緣，零距離沙漠體驗
- 哈密五堡魔鬼城：東疆壯麗雅丹地貌群
- 巴里坤草原：高山草原，哈薩克族牧場生態

## 行前必知

1. **氣候**：日夜溫差極大，即使夏天也要帶防風保暖外套或輕薄羽絨衣
2. **防曬**：氣候乾燥、紫外線強，護唇膏 / 保濕乳液 / 高係數防曬乳為必備
3. **路途**：景點間車程動輒 4–6 小時，備好隨身充電、頸枕、零食
4. **獨庫公路**：每年 6 月中旬至 10 月初開放，出發前確認最新公告
5. **導航**：中國境內自駕用高德地圖（GCJ-02），Google Maps（WGS-84）座標不同勿混用
